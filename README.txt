RESPONDIENDO PREGUNTAS:

¿ Qué tipo de patrón usaste para el proyecto ?

	Para construir todo el sitio use Flexbox.

¿ Por qué decidiste utilizar este patrón y no otro ?

	- Porque es util y sencillo de usar, da los resultados que quiero debido a sus amplias
	posibilidades, float da mas trabajo al usarlo, grid no tiene mucha utilidad para este
	desafio (use grid en mi pagina pero lo minimo para darle algo prescencia), Flexbox es 
	suficiente para este caso.

	- De los elementos de Boostrap, solo use la barra de navegacion para moviles mas que nada
	por la hamburguesa (lo que no quiere decir que no pueda hacerla sin bootstrap, pero hubiera 
	tomado mas tiempo), el resto lo hice con las funciones propias de CSS para no depender tanto 
	de el framework.

¿ Qué tecnología y/o estrategias usarás para que la maqueta sea responsiva ?

	- Usar medidas relativas en la mayoria de los elementos junto a Flexbox y los media querys 
	para ajustar el contenido segun el dispositivo.
